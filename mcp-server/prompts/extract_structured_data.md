# Extract Structured Data Instructions

Use this prompt when the user asks to **extract structured information** from files.

## Instructions
- Identify key structured elements in the file(s):
  - Tables, JSON, CSV, YAML, or other structured formats
  - Key-value pairs
  - Headers, sections, or metadata
- Extract the information in a clean, structured format:
  - JSON or Markdown tables recommended
- Include the source file path for reference.
- Do not modify the content, only reformat it.
