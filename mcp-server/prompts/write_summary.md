# Write Summary Instructions

Use this prompt when the user asks to **summarize files or directories**.

## Instructions
- Read the content of the specified file(s).
- Provide a concise summary capturing:
  - Main purpose of the file or folder
  - Key sections, functions, or content
  - Any notable patterns, repeated structures, or important details
- Keep the summary clear, structured, and easy to understand.
- Mention the source file(s) explicitly.
